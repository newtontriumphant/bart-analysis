# bart-analysis
Hi there! Thanks for taking a gander at my project! It's currently online @ https://bart-analysis.vercel.app/!

This is a BART (Bay Area Rapid Transit) Analysis (Economics Extra Credit) using realtime API and GTFS data from BART, Wiener Linien, and the Munich Verkehrsbund.

Submission for HCTG TransitTime, double dipping with my school Economics Public Good project for extra credit because whyyy not (:

bart-data.py is a file that outputs correct parsed values of all BART stations (I will definitely reuse this in the future!)

api/proxy.js is a replacement for corsproxy.io since vercel is stupid and blocks it.
